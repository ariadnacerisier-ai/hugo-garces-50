# Hugo Garcés 50

Invitación digital premium inspirada en Fórmula 1 para celebrar los 50 años de Hugo Garcés.

## Requisitos

- Node.js 20.19 o superior.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
```

La versión final se genera en `dist/`.

## Personalización pendiente

1. Sustituir el enlace de Google Maps en `index.html`.
2. Conectar el RSVP a Wix Forms, Formspree, Google Sheets o un backend.
3. Reemplazar las imágenes de muestra por artes finales.
4. Revisar la zona horaria en `js/countdown.js`.
5. Sustituir `assets/images/og-image.jpg` por el arte final de WhatsApp.
